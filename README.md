# Git & GitHub Assignment

---

This assignment must be completed individually. You are required to demonstrate basic Git and GitHub operations including commits, branches, merges, and resolving conflicts. Follow the tasks below carefully.

1. **Create a GitHub account.**  
   Sign up for GitHub using your personal email to manage repositories.

2. **Create a new public repository.**  
   Initialize a repository to store your assignment files online.

3. **Make two commits on the main branch.**  
   Add and modify files, committing your changes twice on main.

4. **Create a new branch.**  
   Branch off main to work on features without affecting main.

5. **Make one commit on the new branch.**  
   Add changes specifically on your branch for tracking.

6. **Merge the branch into main.**  
   Integrate your branch changes back into the main branch.

7. **Bonus: Create a merge conflict and resolve it.**  
   Intentionally cause a conflict and demonstrate resolving it correctly.
